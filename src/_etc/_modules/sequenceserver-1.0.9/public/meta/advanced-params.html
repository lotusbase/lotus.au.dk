You can supplement and fine-tune your BLAST search with the following parameters. The following information was obtained from NCBI—should there be any discrepancy between NCBI's documentation and the one provided on Lotus Base, the former has precedence.

General search options

Parameter Accepted value Default value Description/Comment
-evalue Integer (real number) 10 Expectation value (E) threshold for saving hits
-word_size Integer (real number)
  • BLASTN: 11
  • BLASTP: 3
  • BLASTX: 3
  • TBLASTN: 3
  • TBLASTX: 3
 Word size for wordfinder algorithm. Must be ≥2.
-gapopen Integer
  • BLASTN: 5
  • BLASTP: 11
  • BLASTX: 11
  • TBLASTN: 11
  • TBLASTX: Does not support gapped searches.
 Cost to open a gap
-gapextend Integer
  • BLASTN: 2
  • BLASTP: 1
  • BLASTX: 1
  • TBLASTN: 1
  • TBLASTX: Does not support gapped searches.
 Cost to extend a gap
-matrix String BLOSUM62 Scoring matrix name
-threshold Integer (real number)
  • BLASTN: none
  • BLASTP: 11
  • BLASTX: 12
  • TBLASTN: 13
  • TBLASTX: 13
 Minimum word score such that the word is added to the BLAST lookup table. Must be ≥0.
-comp_based_stats String
  • BLASTN: none
  • BLASTP: 2
  • BLASTX: none
  • TBLASTN: 2
  • TBLASTX: none

Use composition-based statistics for BLASTP or TBLASTN

  • D or d—default (equivalent to 2)
  • 0 or F or f—no composition-based statistics
  • 1—Composition-based statistics as in NAR 29:2994–3005, 2001
  • 2 or T or t—Composition-based score adjustment as in Bioinformatics 21:902–911, 2005, conditioned on sequence properties
  • 3—Composition-based score adjustment as in Bioinformatics 21:902–911, 2005, unconditionally

For other programs, this option is not supported.
-num_descriptions Integer (real number) 500

Number of database sequences to show one-line descriptions for.

Incompatbile with max_target_seqs
-num_alignments Integer (real number) 250

Number of database sequences to show alignments for.

Incompatbile with max_target_seqs

Query filtering options

Parameter Accepted value Default value Description/Comment
-seg String
  • BLASTN: none
  • BLASTP: 12 2.2 2.5
  • BLASTX: none
  • TBLASTN: 12 2.2 2.5
  • TBLASTX: 12 2.2 2.5
 Filter query sequence with SEG. Possibilities: yes (to enable), window locut hicut (to enable, conditionally) or no (to disable)
-soft_masking Boolean
  • BLASTN: true
  • BLASTP: false
  • BLASTX: false
  • TBLASTN: false
  • TBLASTX: false
 Apply filtering locations as soft masks
-lcase_masking Flag none Use lower case filtering in query and subject sequence(s)

Query filtering options

Parameter Accepted value Default value Description/Comment
-gilist String none

Restrict search of database to GIs listed in this file. Local searches only.

Incompatible with negative_gilist, seqidlist, remote, subject, and subject_loc.
-seqidlist

Incompatible with gilist, negative_gilist, remote, subject, and subject_loc.
-negative_gilist String none

Restrict search of database to everything except the GIs listed in this file. Local searches only.

Incompatible with gilist, seqidlist, remote, subject, and subject_loc.
-entrez_query String none

Restrict search with the given Entrez query.

Incompatible with remote
-db_soft_mask

Filtering algorithm ID to apply to the BLAST database as soft masking.

Incompatible with subject and subject_loc
-culling_limit Integer

If the query range of a hit is enveloped by that of at least this many higher-scoring hits, delete the hit

Incompatible with best_hit_overhang and best_hit_score_edge.
-best_hit_overhang Real number

Best hit algorithm overhang value (recommended value: 0.1). Value should be between 0 and 0.5.

Incompatible with culling_limit
-best_hit_score_edge Real number

Best hit algorithm score edge value (recommended value: 0.1). Value should be between 0 and 0.5.

Incompatible with culling_limit
-max_target_seqs Integer 500

Maximum number of aligned sequences to keep.

Incompatible with num_descriptions or num_alignments.
