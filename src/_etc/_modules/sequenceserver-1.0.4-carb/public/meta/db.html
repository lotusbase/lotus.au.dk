The databases that are available for searching are organized into several groups for ease of use:

Database name Database type Description
Lotus japonicus Gifu mRNA Nucleotide De novo assembled transcript contigs from Illumina non-directional mRNA-seq data. 58,527 sequences; 40,650,058 total letters.
Lotus japonicus MG20 Illumina- and Sanger-based assembly Nucleotide The program PCAP.rep was used to assemble Illumina contigs (LotusK31ver2GC_illumina_contigs) with the Sanger shotgun data (LjSGA).
Lotus japonicus MG20 PacBio contigs Nucleotide .ctg are the high confidence contigs. PacBio is a sequencing technology that generates long (up to 10 kb) but error-prone (up to 20%) reads. The PacBio reads have been corrected using Illumina data prior to contig assembly.
Lotus japonicus MG20 PacBio unitigs Nucleotide .utg are all the assembled PacBio contigs (unitigs). PacBio is a sequencing technology that generates long (up to 10 kb) but error-prone (up to 20%) reads. The PacBio reads have been corrected using Illumina data prior to contig assembly.
Lotus japonicus MG20 mRNA Nucleotide De novo assembled transcript contigs from Illumina non-directional mRNA-seq data. 57,916 sequences; 39,463,161 total letters.
Lotus japonicus MG20 raw Illumina-based assembly Nucleotide Genomic MG20 contigs assembled using SOAP de novo based on four Illumina libraries with different insert sizes. Raw and uncurated assembly. Contigs may overlap. 904,251 sequences; 439,303,697 total letters.
Lotus japonicus MG20 v3.0 CDS Nucleotide CDS library of Lotus japonicus MG20. 50,599 sequences; 45,983,720 total letters.
Lotus japonicus MG20 v3.0 cDNA Nucleotide cDNA library of Lotus japonicus MG20. 98,043 sequences; 108,691,465 total letters.
Lotus japonicus MG20 genome v2.4 Nucleotide Lotus genome release 2.4. This is the genome version that matches the Apollo browser GFF files in the folder "-Apollo favourite genes/2010_09_06_GFF" on the commom server. According to Shusei, there should only be minor changes to the coordinates on a part of chr5 between version 2.4 and 2.5. There are large differences with respect to the annotation between the two versions due to removal of a lot of retro-element-related sequence annotation in v2.5.
Lotus japonicus MG20 genome v2.5 Nucleotide Lotus genome release 2.5, only sequences anchored on chromosomes, including chr0 long contigs with unknown position and the chloroplast sequence. 8 sequences; 301,492,348 total letters.
Lotus japonicus MG20 genome v2.5 + LjSGA Nucleotide As lj_r25 but including also all the short sanger shotgun contigs (LjSGAs), which are not anchored on chromosomes. 67,748 sequences; 397,315,927 total letters.
Lotus japonicus MG20 genome v2.8 Nucleotide Lotus genome pre-release 2.8. This is not an official release but the current state of the new Lotus genome assembly. Overlaps between contigs still need to be checked, and the coordinates will not match the version to be published. Contains only sequences anchored on the six Lotus chromosomes. 6 sequences; 295,202,327 total letters.
Lotus japonicus MG20 genome v3.0 Nucleotide Version 3.0 of the Lotus genome including chr0 with unanchored contigs, chr1‐6, chloroplast and mitochondrion sequences. 9 sequences; 447,516,816 total letters.
Lotus japonicus probes Nucleotide

The probes used to generate the LjGEA dataset. The naming nomenclature for these probes‐when ending with…

  • _at: Anti-sense target
  • _st: Sense target
  • _s_at: Probeset shares common probes among multiple transcripts (originated from multiple genes)
  • _a_at: Gene may code for alternatively-spliced transcripts
  • _x_at: It was not possible to design unique probes for multiple transcripts (this probeset is not unique for the gene)
Ramularia collo-cygni DK05 contigs Nucleotide Contig library of Ramularia collo-cygni from batch DK05.
DeepSAGE tag sequences Nucleotide Transcript contigs from L. japonicus Gene index 6.0. These TCs were used for annotation of tags from the DeepSAGE experiment. 43,180 sequences; 30,997,719 total letters.
KAW12 scaffolds Nucleotide Genome sequence of the KAW endophyte worked on by Simona and Rafal. 7 sequences; 7,876,853 total letters.
KAW12 translations Nucleotide Amino acid sequence of the endophyte Rhizobium KAW12 genome. 7574 sequences; 2,292,824 total letters.
Lotus japonicus MG20 v3.0 Proteins Protein Protein library of Lotus japonicus MG20. 50,599 sequences; 15,289,260 total letters.
